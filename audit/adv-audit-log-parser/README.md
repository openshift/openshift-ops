This script parses the JSON audit log files generated by OCP/Kubernetes Masters.

Example:
```
# Follow a log file
./ocp-audit.py -f /var/log/ocp-audit.log -F

# Follow a log file and watch for all api calls by user "propyless"
./ocp-audit.py -f /var/log/ocp-audit.log -F -u propyless

# Parse and filter log file for all delete requests by user "PROPYLESS" in projects "default" and "openshift"
./ocp-audit.py -f /var/log/ocp-audit.log -u PROPYLESS -n default -n openshift

# Just parse the log file
./ocp-audit.py -f /var/log/ocp-audit.log
```


```
# ./ocp-audit.py --help
usage: ocp-audit.py [-h] [-c] -f LOGPATH [-F] [-t TIMESTAMP] [-v VERBS]
                    [-u USERS] [-g GROUPS] [-s SOURCEIPS] [-n NAMESPACES]
                    [-r RESOURCES] [-l URI] [-p] [--requests | --responses]
Python App to parse and filter OCP audit logs in JSON format.
optional arguments:
  -h, --help     show this help message and exit
  -c             Count the number of unique hits based on VERB, USER, GROUP,
                 RESOURCE, NAMESPACE
  -f LOGPATH     Log file to load
  -F             Follow log file
  -t TIMESTAMP   Which time to search
  -v VERBS       Which verbs to search for "get, list, watch, create, update,
                 patch, delete, deletecollection, and proxy"
  -u USERS       names of users to search for.
  -g GROUPS      names of groups to search for.
  -s SOURCEIPS   IP(s) to search for.
  -n NAMESPACES  Namespaces to search for.
  -r RESOURCES   Resources to search for.
  -l URI         URI to search for.
  -p             Raw log format
  --requests     Filter for Requests received
  --responses    Filter for Responses transmitted
```
